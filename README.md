## 🚀 Featured Projects

### 🩺 [HealthMate AI](https://github.com/rachitxcodes/Healthmate-AI)
Vision-LLM OCR pipeline (Gemini) that turns unstructured lab report images into structured clinical profiles at 95%+ accuracy — feeding calibrated risk classifiers (RF, XGBoost) and a live IoT (ESP32) health fusion engine, served via FastAPI + React.

### 📉 [Telecom Customer Churn Prediction](https://github.com/rachitxcodes/Customer-Churn-Prediction)
EDA on 7,000+ records to isolate top churn drivers, benchmarked 5 classifiers with SMOTEENN balancing (best: tuned Logistic Regression, Recall 87.7%, ROC-AUC 0.83), tracked with MLflow/SHAP/Optuna, and shipped as a Streamlit dashboard with a Gemini-powered retention advisor.

### 🚨 [LifeGuard Band](https://github.com/rachitxcodes/LifeGuard-Band)
Wearable fall-detection device (ESP32 + MPU6050) using threshold-calibrated accelerometer classification to trigger a GSM SOS alert with live GPS coordinates for emergency response.
